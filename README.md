# Smart-Eval AI

Next-Generation Automated Exam Grading & Feedback System

## Project Overview

Smart-Eval AI is a web-based platform designed to revolutionize the evaluation process of engineering examination papers by integrating Artificial Intelligence (OCR & LLM) with a Full-Stack web architecture.

## Project Structure

```
Project 2/
├── Project_Docs/          # Project documentation
├── smart-eval-frontend/   # React.js frontend application
├── smart-eval-backend/    # Flask backend application
├── docker-compose.yml     # Production Docker configuration
├── docker-compose.dev.yml # Development Docker configuration
└── README.md             # This file
```

## Technology Stack

### Frontend
- React.js 18
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Flask (Python 3.11)
- MongoDB
- Redis
- Celery
- Local Vision (LM Studio) for OCR
- Local LLM (LM Studio) for grading
